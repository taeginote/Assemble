### 1차 설계

#### x.x.x

- 회원가입, 로그인 (JWT)

- 모임 목록

  - 모임 목록 (최신순, 인기순, 좋아요 순 정렬)
  - 모임 검색 (제목, 내용, 작성자)

- 마이페이지

  - 내가 작성한 글
  - 내가 작성한 댓글
  - 회원 탈퇴

- 모임 CRUD

- 댓글 CRUD

- 좋아요 추가, 삭제

<br/>

### 2차 설계 예정

- 모임 고도화
- 마이 페이지 기능 추가
- 관리자
- 실시간 검색어
- 자유 게시판
- 로그인 OAuth
